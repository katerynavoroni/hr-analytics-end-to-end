# HR Analytics End-to-End Project

## Overview

This project presents an end-to-end HR analytics workflow using SQL, Python, and Power BI.

The goal was to analyze workforce data, clean and transform datasets, calculate key HR KPIs, and build an interactive dashboard for business insights.

---

## Project Workflow

1. Data extraction and analysis in SQL
2. Data cleaning and transformation in Python (Pandas)
3. Data visualization in Python (Matplotlib)
4. Dashboard building in Power BI

---

## Tools Used

* SQL
* Python
* Pandas
* Matplotlib
* Power BI
* DAX
* Power Query

## Dataset

The project uses three related HR datasets:

### employees.csv

Contains employee details:

* employee_id
* department
* position

### salaries.csv

Contains payroll data:

* employee_id
* salary
* bonus

### attendance.csv

Contains attendance data:

* employee_id
* absence_days
* hours_worked
* month
