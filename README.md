# 📊 HR Analytics Dashboard

A Python-based HR analytics dashboard built with **pandas** and **matplotlib**.

## 🔧 Technologies

* Python
* pandas
* matplotlib

## 📈 KPIs

* Average salary by department
* Total bonus by department
* Average absence days
* Business insights panel

## 📷 Dashboard Preview

![HR Dashboard](output/hr_dashboard.png)

## 🚀 How to run

```bash
pip install -r requirements.txt
python src/main.py
```
